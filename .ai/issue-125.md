# Add INI config merger utility with validation and CLI

Добавить модуль src/inimerge.py.

Нужно реализовать работу с конфигурациями в простом INI-подобном формате.

Формат входных файлов:
- строки вида key=value
- пустые строки игнорируются
- строки, начинающиеся с #, считаются комментариями и игнорируются
- пробелы вокруг key и value нужно обрезать
- если в файле одна и та же key встречается несколько раз, использовать последнее значение

Реализовать функции:

1. parse_ini(path: str) -> dict[str, str]
   - читает файл
   - возвращает словарь key -> value
   - игнорирует пустые строки и комментарии
   - игнорирует невалидные строки без символа =
   - если key пустой после trim, строка игнорируется

2. merge_ini(base_path: str, override_path: str) -> dict[str, str]
   - читает два файла через parse_ini
   - возвращает merged dict
   - значения из override_path должны перезаписывать base_path

3. validate_config(data: dict[str, str]) -> list[str]
   Проверки:
   - если есть ключ port, он должен быть целым числом от 1 до 65535
   - если есть ключ debug, он должен быть только true или false
   - если есть ключ host, он не должен быть пустым
   - функция возвращает список строк-ошибок
   - если ошибок нет, вернуть []

CLI:
python -m src.inimerge --base base.ini --override override.ini

Поведение CLI:
- объединяет конфиги
- валидирует merged config
- если ошибок нет, печатает merged config в stdout
- если ошибки есть:
  - печатает строки ошибок в stderr
  - завершает процесс с кодом 1

Формат stdout (если ошибок нет):
- отсортировать ключи по алфавиту
- выводить строго:
key=value

по одной строке на пару

Пример:
debug=false
host=localhost
port=8080
